My Role and Completed Tasks

Role: As Student 2, I was responsible for designing and implementing the following core pages:

  • Home Page
  • Global CSS
  • Table Page
  • Content Page

Technical Descriptions

Home Page

  • Clean layout using semantic tags and flexbox for header/navigation.
  • Google Fonts applied for improved readability.
  • Responsiveness ensured via max-widths and media queries.

Table Page

  • Used <table> elements with thead and tbody for structure.
  • Used tooltips with icons, merged rows with rowspan, styled hover states.
View Table Page
Validation: Table Page

Content Page

  • Multi-section layout for facts, causes, and sources.
  • Used CSS grid for responsiveness and iframe to embed videos.
  • Each fact is a styled card with hover effects.
View Content Page
Validation: Content Page

Challenges & Lessons Learned

  • Working with rowspan and colspan for mobile table layouts was challenging.
  • Ensuring consistent internal linking using anchor tags and scroll behavior across browsers was a learning experience.

Accessibility

Used semantic HTML, alt text for all images, descriptive titles for links, and ensured keyboard-navigable elements.

Compliance

All pages follow valid HTML5, CSS3, and Janet web standards. File names use lowercase and avoid spaces.
